# Mock Test Analytics Dashboard

A JavaScript-based analytics dashboard that helps students analyze mock test performance, accuracy, weak subjects, and negative marking trends.

## 🚀 Features
- Add mock test performance
- Accuracy calculation
- Negative marking tracking
- Interactive performance chart
- LocalStorage data persistence
- Dynamic dashboard statistics

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript
- Chart.js

## 📚 Concepts Used
- DOM Manipulation
- LocalStorage
- Event Handling
- Arrays & Objects
- Chart.js
- Data Analytics Logic

## ▶️ How to Run
Open `index.html` in browser.
